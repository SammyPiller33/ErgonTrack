# Task Tracker CLI

[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com/votre-username/ErgonTrack)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

A simple command-line task manager written in Python.

## Description

Task Tracker CLI is a command-line application that allows you to manage your daily tasks. You can add, modify, delete, and track the status of your tasks directly from your terminal.

## Features

- Add new tasks
- Edit existing tasks
-  Delete tasks
- Display task list
- Change task status
- Automatic JSON storage

## Prerequisites

- Python 3.14.0 or higher
- virtualenv

## Usage

Run the application with: `python main.py`
